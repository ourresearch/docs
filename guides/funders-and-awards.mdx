---
title: Funders and awards
description: Understanding research funding in OpenAlex
---

Funders are organizations that fund research. Awards are specific grants or funding decisions. OpenAlex tracks funding acknowledgments across millions of works.

## Funders

```bash
# List all funders
https://api.openalex.org/funders

# Get a specific funder
https://api.openalex.org/funders/F4320306076
```

### Key properties

| Property | Description |
|----------|-------------|
| `id` | OpenAlex ID (e.g., `F4320306076`) |
| `ror` | ROR identifier |
| `display_name` | Funder name |
| `country_code` | ISO country code |
| `works_count` | Funded works |
| `cited_by_count` | Citations to funded works |
| `grants_count` | Number of grants |
| `description` | Funder description |

For a comprehensive field listing, see the [Get a single funder](/api-reference/funders/get-a-single-funder) response specification.

### Common funder filters

| Filter | Example |
|--------|---------|
| By country | `filter=country_code:US` |
| By continent | `filter=continent:Q46` |
| Global South | `filter=is_global_south:true` |
| Has ROR | `filter=has_ror:true` |
| High funding | `filter=awards_count:>1000` |

```bash
# US government funders
https://api.openalex.org/funders?filter=country_code:US&sort=-works_count

# Funders in Europe
https://api.openalex.org/funders?filter=continent:Q46
```

## Awards

Awards represent specific grants or funding decisions linked to works.

```bash
# List all awards
https://api.openalex.org/awards

# Get a specific award
https://api.openalex.org/awards/https://doi.org/10.35802/123456
```

### Key properties

| Property | Description |
|----------|-------------|
| `id` | Award identifier |
| `funder` | Funding organization |
| `award_id` | Grant number |
| `start_year` | Grant start year |
| `end_year` | Grant end year |
| `amount` | Funding amount |
| `currency` | Amount currency |
| `funded_outputs_count` | Works acknowledging this grant |
| `lead_investigator` | Principal investigator |

For a comprehensive field listing, see the [Get a single award](/api-reference/awards/get-a-single-award) response specification.

### Common award filters

| Filter | Example |
|--------|---------|
| By funder | `filter=funder.id:F4320306076` |
| By year | `filter=start_year:2023` |
| By investigator | `filter=lead_investigator.orcid:0000-0002-1234-5678` |
| High output | `filter=funded_outputs_count:>10` |

```bash
# NIH awards from 2023
https://api.openalex.org/awards?filter=funder.id:F4320306076,start_year:2023
```

## Finding funded works

```bash
# Works funded by NIH
https://api.openalex.org/works?filter=grants.funder:F4320306076

# Works acknowledging a specific grant
https://api.openalex.org/works?filter=grants.award_id:R01-GM123456
```
