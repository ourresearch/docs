This page tracks significant updates to the OpenAlex API and documentation.

## 2026

### February 13, 2026
**API keys required** — All API requests now require an API key. [Get your free key](https://openalex.org/settings/api).

### February 4, 2026
**API keys recommended** — API keys are now recommended for all requests. Starting Feb 13, they become required.
