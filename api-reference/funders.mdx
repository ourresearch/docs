Funders are organizations that fund research. OpenAlex indexes about 32,000 funders. Funder data comes from Crossref, and is enhanced with data from Wikidata and ROR.

Funders are connected to works through [grants](/guides/key-concepts#grants).

## Fields

<ResponseField name="id" type="string">
  The OpenAlex ID for this funder.

  `"https://openalex.org/F4320332161"`
</ResponseField>

<ResponseField name="display_name" type="string">
  The primary name of the funder.

  `"National Institutes of Health"`
</ResponseField>

<ResponseField name="alternate_titles" type="array">
  A list of alternate titles for this funder.

  `["US National Institutes of Health", "NIH"]`
</ResponseField>

<ResponseField name="country_code" type="string">
  ISO two-letter country code where this funder is located.

  `"US"`
</ResponseField>

<ResponseField name="description" type="string">
  A short description from Wikidata.

  `"medical research organization in the United States"`
</ResponseField>

<ResponseField name="homepage_url" type="string">
  URL for this funder's primary homepage.

  `"http://www.nih.gov/"`
</ResponseField>

<ResponseField name="image_url" type="string">
  URL for an image representing this funder (usually from Wikimedia).
</ResponseField>

<ResponseField name="image_thumbnail_url" type="string">
  Smaller version of `image_url`.
</ResponseField>

<ResponseField name="works_count" type="integer">
  The number of works linked to this funder.

  `260210`
</ResponseField>

<ResponseField name="cited_by_count" type="integer">
  Total citations to works linked to this funder.

  `7823467`
</ResponseField>

<ResponseField name="grants_count" type="integer">
  The number of grants linked to this funder.

  `7109`
</ResponseField>

<ResponseField name="ids" type="object">
  External identifiers for this funder:
  - `openalex`: OpenAlex ID
  - `crossref`: Crossref Funder ID
  - `ror`: ROR ID
  - `wikidata`: Wikidata ID
  - `doi`: DOI
</ResponseField>

<ResponseField name="roles" type="array">
  Links to related entities when an organization has multiple roles (funder, publisher, institution).
</ResponseField>

<ResponseField name="summary_stats" type="object">
  Citation metrics:
  - `2yr_mean_citedness`: Impact factor
  - `h_index`: h-index
  - `i10_index`: i-10 index
</ResponseField>

<ResponseField name="counts_by_year" type="array">
  `works_count` and `cited_by_count` for each of the last ten years.
</ResponseField>

<ResponseField name="created_date" type="string">
  When this funder was added to OpenAlex (ISO 8601 date).
</ResponseField>

<ResponseField name="updated_date" type="string">
  When this funder was last updated (ISO 8601 datetime).
</ResponseField>
