Work types classify the form of a scholarly document. Each work has exactly one type.

## Values

| ID | Display name |
|----|-------------|
| `article` | article |
| `book` | book |
| `book-chapter` | book-chapter |
| `dataset` | dataset |
| `dissertation` | dissertation |
| `editorial` | editorial |
| `erratum` | erratum |
| `letter` | letter |
| `libguides` | libguides |
| `other` | other |
| `paratext` | paratext |
| `peer-review` | peer-review |
| `preprint` | preprint |
| `reference-entry` | reference-entry |
| `report` | report |
| `retraction` | retraction |
| `review` | review |
| `standard` | standard |
| `supplementary-materials` | supplementary-materials |

Use these values with the `type` filter on works: `filter=type:article`

See the [Works guide](/api-reference/works) for details on how work types are assigned.
