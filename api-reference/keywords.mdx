Keywords are short words or phrases assigned to works using an automated system based on Topics.

## Fields

<ResponseField name="id" type="string">
  The OpenAlex ID for this keyword.

  `"https://openalex.org/keywords/cardiac-imaging"`
</ResponseField>

<ResponseField name="display_name" type="string">
  The English-language label of the keyword.

  `"Cardiac Imaging"`
</ResponseField>

<ResponseField name="works_count" type="integer">
  The number of works tagged with this keyword.

  `21737`
</ResponseField>

<ResponseField name="cited_by_count" type="integer">
  The total citations to works tagged with this keyword.

  `4347000`
</ResponseField>

<ResponseField name="created_date" type="string">
  When this keyword was added to OpenAlex (ISO 8601 date).

  `"2024-04-10"`
</ResponseField>

<ResponseField name="updated_date" type="string">
  When this keyword was last updated (ISO 8601 datetime).

  `"2024-05-09T05:00:03.798420"`
</ResponseField>
