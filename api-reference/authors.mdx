Authors are people who create works. OpenAlex indexes tens of millions of disambiguated author profiles.

## Filter, sort, and group_by fields

The API supports filtering, sorting, and grouping on authors fields. See [Filtering](/guides/filtering), [Sorting](/guides/sorting-and-paging), and [Grouping](/guides/grouping) for syntax.
